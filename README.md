# Samsung E-Commerce Landing Page Clone

A responsive landing page project inspired by the Samsung official website. This project showcases a modern e-commerce UI with complex navigation, product sliders, and interactive elements.

##  Features

- **Interactive Mega Menu:** A detailed multi-column dropdown menu system for various categories like Shop, Mobile, TV & Audio, and more.
- **Hero Slider:** An immersive full-screen carousel using **Swiper.js** to highlight featured products and deals.
- **Responsive Product Grids:** Organized layouts for product categories using CSS Grid and Flexbox.
- **Dynamic Tabbed Sliders:** A custom tabbed section for "Deals on last-minute gifts" that allows users to switch between product categories (Mobile, TV, Appliances).
- **Smooth Navigation:** Includes a "Back to Top" button and smooth scrolling behavior.
- **Visual Feedback:** Interactive UI elements with hover effects, including a background dimming effect when navigating the menu.

##  Technologies Used

- **HTML5:** Semantic structure for the web pages.
- **CSS3:** Custom styling including:
  - **CSS Grid & Flexbox** for layouts.
  - **CSS Variables** for dynamic theme management.
  - **Animations & Transitions** for smooth UI interactions.
- **Bootstrap 5.3.8:** Used for grid utility and baseline styling components.
- **Swiper.js (v11/12):** Utilized for the main hero slider and product carousels.
- **Font Awesome 7.0.1:** For high-quality vector icons.
- **Google Fonts / Custom Fonts:** Implementation of the "SamsungSharpSans" brand typography.
